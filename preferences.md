# Job Search Preferences

## Target roles
- Graduate / early-career roles in data science or machine learning
- Not fussy on exact title — open to any role that fits the skill set below (data, numbers, computing). Examples: Data Analyst, Data Scientist, ML Engineer, Quantitative Analyst, Research Analyst, Data Engineer.
- Seniority: graduate / entry-level / early-career only (not requiring years of prior industry experience)

## Location
- London, UK
- Preference order: on-site > hybrid > remote
- Not interested in fully remote-only roles unless exceptional

## Company size / type
- Bias toward mid-sized companies
- Avoid early-stage startups (failure risk) and very large corporations (high-volume applicant pools, e.g. 1000+ applicants per role)

## Compensation
- Minimum £35,000 per annum — treat as a soft floor, flag if a strong match falls slightly short

## Industry
- No industry preference — open to any sector. Fit is driven by the day-to-day tasks (data/numbers/computing-heavy work), not the company's end product or mission

## Dealbreakers
- None specified beyond the above (company size, location, salary floor)

## Notes for the agent
- Skills to match against: Python, C++, data modelling, statistical modelling/numerical methods, neural networks/ML, SQL-adjacent/data analysis work, API integration, web app development
- Tobias is open-minded on role title — prioritise matching the actual day-to-day work over keyword-matching job titles
